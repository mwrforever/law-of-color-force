# 《色力法则》核心系统设计决策确认（2026-07-15）

基于 `design/gdd/` 四份 GDD，以下设计分叉点经用户确认，作为 engineering-lead 落地代码的**基准**。所有数值为起步建议，真机手感微调时由工程负责人回传更新。

## 已确认决策

1. **三原色同体 → ⚪ 白中和重置**：`ColorResolver` 中 `{R,B,Y}` → `WHITE`，物理回中性（`m_g=1.0, a_b=0, e=0.20, DYNAMIC`）。给玩家非橡皮擦的紧急脱困手段。
   - 对应：color-physics-gdd §4
2. **覆盖模型 → 叠加原色栈 + LIFO 橡皮擦**：喷涂幂等压栈（已存在移至栈顶），橡皮擦按最近优先弹出，全局态解析（非区域混合）。稳定构造复合色、可逐层回退不卡死。
   - 对应：color-physics-gdd §5
3. **多人同步权威 → host 客户端权威**：房主客户端算权威，自托管 WebSocket 中继仅转发；客户端插值/预测、弱网重连、关键分值 host 校验。
   - 对应：multiplayer-gdd（技术侧 ADR-004 已定自托管 ws）
4. **自我染色时限 → 3s 持续 + 1s 冷却**：到期自动还原中性；紫色自染强制 `min(3.0, freezeCap=2.5s)`。
   - 对应：color-physics-gdd §6
5. **多人默认模式 → 皇冠争夺（持有皇冠累计 15s 胜）为主，竞速为备选**；超时按持冠秒数 / 进度排名。
   - 对应：multiplayer-gdd
6. **颜料预算 → 单人关 30 次 / 多人每人 20 次**（起步值，真机微调）。
   - 对应：spray-interaction-gdd / level-system-gdd

## 待真机微调项（由 engineering-lead 回传）
- 重力倍率上限 `m_g=2.0`、浮力 `a_b=1.5`、弹性 `e=0.88` 等手感数值。
- 快照频率 15–20Hz 实测取舍。
- 喷涂半径 0.6m、颜料预算数值。
- 自染时限/冷却/紫定身上限。
